# Election Analysis

## Project Overview

The Colorado Board of Elections requested an audit to be performed on election results of a particular race. I was instructed to use Python to create a text file that will produce the follwoing information. 
1. Total number of votes cast
2. A complete list of candidates who received votes
3. Total number of votes each candidate received
4. Percentage of votes each candidate won
5. The winner of the election based on popular vote

## Summary 

1. Total Votes
    - 369,711
2. County Names
    - Jefferson
    - Denver
    - Arapahoe
3. Candidates Names
    - Charles Casper Stockham
    - Diana DeGette
    - Raymon Anthony Doane
4. Number and percentage of votes per county
    - Jefferson, 38,855 votes, 10.5% of total votes
    - Denver, 306,055 votes, 82.8% of total votes
    - Arapahoe, 24,801 votes, 6.7% of total votes
5. Largest County Turnout
    - Denver
6. Number and percentage of votes per candidate
    - Charles Casper Stockham, 85,213 votes, 23.0% of total votes
    - Diana DeGette, 272,892 votes, 73.8% of total votes
    - Raymon Anthony Doane, 11,606 votes, 3.1% of votes
7. Winner of congressional election:
    - Diana DeGette
